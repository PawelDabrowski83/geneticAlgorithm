# Genetic algorithm

Simple implementation of genetic algorithm.

START
<ul>
<li>Generate the initial population</li>
<li>Compute fitness</li></ul>
REPEAT
<ul>
<li>Selection</li>    
    <li>Crossover</li>
    <li>Mutation</li>
    <li>Compute fitness</li></ul>
UNTIL population has converged<br/>
STOP
